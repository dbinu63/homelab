# homelab

This repository documents a personal homelab built around a NAS (Network Attached Storage) system using Debian, OpenMediaVault, Docker, and Tailscale.  
The system centralizes Blender projects and other files, providing cross-device access while demonstrating hands-on Linux, networking, and containerization skills.

## Objectives

- Centralize file storage across multiple devices
- Enable secure remote access from anywhere
- Experiment with containerized services using Docker
- Build a foundation for a scalable homelab environment

## Current Setup

- **NAS:** Debian-based server with OpenMediaVault  
- **Containers:** Docker managed via Portainer for additional services  
- **Remote Access:** Tailscale VPN for secure connections  
- **File Sync:** Planned implementation with Syncthing  

## Use Case

- Store and manage large Blender project files
- Access files across multiple machines (PC and MacBook)
- Enable future distributed rendering workflows

## Lessons Learned

- Linux system setup and configuration
- Remote networking and VPN setup
- Docker container management
- Centralized storage and workflow optimization

## Future Improvements

- Automated file synchronization with Syncthing
- Distributed rendering management via Flamenco
- Backup strategies and monitoring tools for reliability
